dagworker — DAG workflow orchestrator tool.

Actions:
- `start`: Create and fork a workflow; returns `workflowId` immediately; execution runs in background.
- `status`: Poll a workflow's real-time state — pending/running/completed/failed nodes, violations, duration.
- `cancel`: Abort a running workflow.
- `pause`: Pause a running workflow — running nodes continue but no new nodes are scheduled.
- `resume`: Resume a paused workflow — status returns to running and ready nodes are scheduled.
- `list`: See every workflow spawned in the lifetime of this process.
- `replan`: Restructure a running workflow's tail (add/remove/update pending nodes, optionally change max_concurrency). Terminal/frozen nodes are immutable scaffolding. Writes one atomically-applied row to dag_workflow_history.
